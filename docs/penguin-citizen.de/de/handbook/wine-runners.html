    Wine Runners

    Wine/Proton Runner herunterladen und verwalten, DXVK-Versionen und Wine-Prefix-Tools.

    Installierte Runner

    Der obere Bereich listet alle aktuell auf deinem System installierten Wine/Proton Runner auf. Jeder Runner zeigt:

    • Runner-Name und Version
    • Installationspfad
    • Ob er vom aktuell aktiven Launch-Profil genutzt wird (hervorgehoben)
    • Nutzungs-Badges — Used by N profile(s) (Hover zeigt die Profilnamen) und Fallback wenn dieser Runner als globaler Wine-Runner-Fallback gesetzt ist

    Der Aktivieren-Button setzt einen Runner fuer das aktive Profil. Der Runner ist jetzt eine profil-spezifische Einstellung — ein Wechsel hier markiert das aktive Profil als Dirty, bis du den neuen Stand via Update Profile auf der Startseite uebernimmst (oder einfach so weiter spielst).

    Der Loeschen-Button entfernt einen Runner von deinem System. Wenn der Runner von einem oder mehreren Profilen genutzt wird, listet der Bestaetigungsdialog diese Profile plus den Fallback-Hinweis auf, damit du nicht versehentlich ein funktionierendes Setup zerlegst.

    Fuer den taeglichen Runner-Wechsel nimm das Runner-Dropdown direkt in der Profil-Leiste der Startseite — kein Umweg ueber diese Seite noetig. Die Wine-Runner-Seite ist fuer Installation, Loeschen und Bulk-Verwaltung von Runnern.

    Verfuegbare Runner

    Durchsuche und lade Runner aus mehreren Community-Quellen herunter:

    • LUG Helper - Von der Star Citizen Linux Users Group kuratierte Runner. Dies ist die primaere und empfohlene Quelle.
    • Kron4ek - Wine-GE Builds fuer allgemeine Kompatibilitaet.
    • RawFox - Spezialisierte Wine Builds.
    • Mactan - Community-gepflegte Runner-Builds.

    Jeder verfuegbare Runner zeigt seinen Namen, Version und Veroeffentlichungsdatum. Klicke auf Installieren, um ihn herunterzuladen und zu entpacken. Ein Fortschrittsbalken zeigt den Download-Status. Bereits installierte Runner sind entsprechend markiert.

    Verwende den Aktualisieren-Button, um die neuesten Runner-Listen von GitHub abzurufen. Runner-Listen werden eine Stunde lang zwischengespeichert, um uebermassige API-Aufrufe zu vermeiden.

    Wenn du einen GitHub-Token in den Einstellungen hinterlegst, erhaeltst du hoehere API-Rate-Limits beim Durchsuchen verfuegbarer Runner. Das ist nuetzlich, wenn du haeufig nach Updates suchst.

    DXVK-Verwaltung

    DXVK uebersetzt DirectX-Aufrufe in Vulkan, was fuer den Betrieb von Star Citizen unter Linux unerlasslich ist. Dieser Bereich zeigt:

    • Die aktuell installierte DXVK-Version
    • Verfuegbare Versionen von GitHub Releases
    • Einen Installieren/Aktualisieren-Button zum Herunterladen und Bereitstellen der neuesten Version

    DXVK-DLLs werden waehrend der Installation automatisch an den richtigen Stellen im Wine-Prefix platziert.

    Prefix-Tools

    Schnellzugriff auf gaengige Wine-Prefix-Werkzeuge:

    • Winecfg - Oeffnet den Wine-Konfigurationsdialog. Nuetzlich zum Anpassen der DPI-Skalierung, Windows-Version und anderer Wine-Einstellungen.
    • Wine Shell - Oeffnet eine Terminal-Sitzung innerhalb des Wine-Prefix. Hilfreich fuer Debugging und manuelle Konfiguration.
    • PowerShell - Installiert PowerShell Core im Wine-Prefix ueber winetricks. Einige Tools und Skripte benoetigen dies. Zeigt Download-Fortschritt und Installationslogs.
    Runner-Quellen werden automatisch verwaltet. Penguin Citizen importiert und aktualisiert die LUG Helper-Quellenliste bei jedem Seitenaufruf, um sicherzustellen, dass du immer die neuesten empfohlenen Runner siehst.
