Privacy Policy

Privacy policy for the Penguin Citizen website

1. Controller

Responsible for data processing on this website: Thomas Griwenka Avenida Touroperador Finnmatkat 10 ES, 35100 Maspalomas Email: tomrhodan@gmail.com

2. Collection and Storage of Personal Data (Server Log Files)

The provider of this website automatically collects and stores information in so-called server log files, which your browser automatically transmits to the server. These are typically:

  • Browser type and browser version
  • Operating system used
  • Referrer URL
  • Hostname of the accessing computer
  • Time of the server request
  • IP address

This data is not merged with other data sources. The legal basis for this data processing is Art. 6 (1) (f) GDPR, which allows the processing of data based on our legitimate interest in the technically error-free presentation and optimization of our website.

3. Cookies and Tracking

This website strictly respects your privacy. It does not set any cookies, does not use any tracking services (like Google Analytics), and does not transmit data to third parties for advertising purposes. Local browser storage (localStorage) is used exclusively for technical purposes (a five-minute version cache); no personal data is stored.

4. Automatic Version Lookup

When the homepage loads, this site queries the public GitHub API (api.github.com) to display the current program version and the download links for the release artefacts. Your IP address is transmitted to GitHub Inc. (Microsoft, USA) as part of this request. The legal basis is Art. 6(1)(f) GDPR (legitimate interest in showing visitors current version information). The response is cached in your browser's local storage for five minutes to avoid repeated requests. GitHub privacy notice: https://docs.github.com/site-policy/privacy-policies

5. External Links

This website contains links to external websites (e.g., GitHub). I am not responsible for the privacy practices or the content of these external providers.

6. Your Rights

Under the GDPR, you have the right to request information about your stored personal data, its origin, recipients, and the purpose of its collection at any time and free of charge. You also have the right to request the correction, blocking, or deletion of this data. Since this website only temporarily processes IP addresses via standard server logs, please note that technically, no permanently identifiable data is retained. You can contact me at any time at the address given above if you have further questions on the subject of data protection.
