Handbook

    Wine Runners

    Download and manage Wine/Proton runners, DXVK versions, and Wine prefix tools.

    Installed Runners

    The top section lists all Wine/Proton runners currently installed on your system. Each runner shows:

    • Runner name and version
    • Installation path
    • Whether it is the runner used by the currently active Launch Profile (highlighted)
    • Usage badges — Used by N profile(s) (hover for the list of profile names) and Fallback if this runner is set as the global Wine Runner Fallback

    The Activate button sets a runner for the active profile. Note that the runner is now a per-profile setting — switching here puts the active profile into the dirty state until you commit the change via Update Profile on the Launch page or just keep launching with it.

    The Delete button removes a runner from your system. If the runner is referenced by one or more profiles, the confirmation dialog lists those profiles plus the fallback hint so you don't accidentally break a working setup.

    For day-to-day runner switching, use the runner dropdown directly on the Launch page's profile bar — no roundtrip through this page needed. The Wine Runners page is for installing, deleting, and bulk-managing runners.

    Available Runners

    Browse and download runners from multiple community sources:

    • LUG Helper - Runners curated by the Star Citizen Linux Users Group. This is the primary and recommended source.
    • Kron4ek - Wine-GE builds for general compatibility.
    • RawFox - Specialized Wine builds.
    • Mactan - Community-maintained runner builds.

    Each available runner shows its name, version, and release date. Click Install to download and extract it. A progress bar shows the download status. Runners that are already installed are marked accordingly.

    Use the Refresh button to re-fetch the latest runner lists from GitHub. Runner lists are cached for one hour to avoid excessive API calls.

    If you set a GitHub token in Settings, you get higher API rate limits when browsing available runners. This is useful if you frequently check for updates.

    DXVK Management

    DXVK translates DirectX calls to Vulkan, which is essential for running Star Citizen on Linux. This section shows:

    • The currently installed DXVK version
    • Available versions from GitHub releases
    • An Install/Update button to download and deploy the latest version

    DXVK DLLs are automatically placed in the correct Wine prefix locations during installation.

    Prefix Tools

    Quick access to common Wine prefix utilities:

    • Winecfg - Opens the Wine configuration dialog. Useful for adjusting DPI scaling, Windows version, and other Wine settings.
    • Wine Shell - Opens a terminal session inside the Wine prefix. Helpful for debugging and manual configuration.
    • PowerShell - Installs PowerShell Core into the Wine prefix via winetricks. Some tools and scripts may require this. Shows download progress and installation logs.
    Runner sources are managed automatically. Penguin Citizen imports and updates the LUG Helper source list on each page load to ensure you always see the latest recommended runners.
