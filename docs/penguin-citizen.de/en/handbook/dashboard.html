Handbook

    Dashboard

    Your command center with live Star Citizen news, server status, and community statistics.

    Status Cards

    The top row displays three status cards that give you a quick overview:

    • Star Citizen Installation - Shows your install path and detected language. A yellow indicator means a translation update is available; green means everything is current. Use the "Open Folder" button to quickly access the installation directory. The "Repair Installation" button starts the repair function (see below).
    • Wine Runner - Displays the runner used by the active Launch Profile and its status (ready, not configured, or missing).
    • Launch - The play button starts and stops Star Citizen directly from the Dashboard — no page jump. The card walks through three states (see below) so a casual player never has to leave the Dashboard.

    Launch Card States

    The launch card is the centerpiece of the Dashboard for casual players. It morphs depending on context:

    • Not installed — disabled "Launch Star Citizen" button with a "complete install first" hint. Run through the Installation page before continuing.
    • No usable profile — primary ⚙ Set up & Launch button opens the Profile Wizard. Reached when no Launch Profile exists yet, the active profile has no runner, or the active profile's runner has been uninstalled.
    • Starting — the button shows a spinner and "Starting…" while launch_game spins up. The Launch page log is still reachable from the sidebar if you want to see the live launch output.
    • Running — the card switches to a green accent stripe with "⦁ Star Citizen is running" and a red ⏹ Stop button. Click Stop to send SIGTERM (with SIGKILL fallback after 2 seconds). The active profile's name stays visible below.
    • Ready — primary "▶ Launch Star Citizen" button + "Profile: <name>" meta line + a small ⚙ Profile Wizard button below for creating additional profiles without leaving the Dashboard.
    The Profile Wizard is a single-step modal: pick a Wine runner (the first lug-… build in your installed set is recommended automatically), name the profile, then choose Save & Done (back to Dashboard) or Customize (jump to the Launch page for fine-tuning). Wizard profiles default to wayland=false for stability — flip Wayland on later via Customize if you want it.

    Installation Repair

    If the RSI Launcher no longer starts correctly or is corrupted, Penguin Citizen offers a one-click repair function. The "Repair Installation" button on the installation status card performs the following steps:

    1. The current Wine prefix is renamed to a backup (e.g., star_citizen_repair_backup_20260406_143022).
    2. The installation wizard is launched to perform a fresh installation.
    3. After reinstallation, existing Star Citizen game data (the StarCitizen folder) is automatically restored from the backup.

    Repair Backup Banner

    When a repair backup exists, an info banner appears above the status cards. It shows the backup path and size. Use the "Delete Backup" button to remove the backup after a successful repair to free up disk space. A confirmation dialog appears before deletion.

    RSI News Feed

    The news panel fetches live articles from the Roberts Space Industries website. Each article shows a title, category, timestamp, and brief summary. Click any article to open it in your browser.

    If the news feed fails to load (e.g., no internet connection), a retry button appears to refresh the feed.

    Server Status

    The server status panel monitors the Star Citizen infrastructure and shows the operational status of key components. Status indicators are color-coded:

    • Green - Operational
    • Yellow - Degraded performance
    • Red - Major outage
    • Gray - Unknown

    Community Statistics

    This panel shows real-time community metrics:

    • Total Funding - The total amount pledged by the community
    • Star Citizens - Total number of registered players
    • Vehicles - Number of vehicles in the game

    Below the current numbers, sparkline charts show historical trends over 7-day and 30-day periods. Use the period toggle buttons to switch between views. The delta percentage shows how much each metric has changed.

    All Dashboard data loads in parallel for fast startup. If any panel fails to load, it shows an error state with a retry button - the other panels remain unaffected.
